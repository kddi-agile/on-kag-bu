## 使用例

- チームプレゼンテーションでのインナーソース概念の説明
- 社内イベントでのポスターやバナー
- ドキュメントの挿絵
- チャットやメッセージでのリアクション画像
- ブログ記事の挿絵
- プルリクエストやコードレビューでのLGTMスタンプ
