## 使用方法

これらのイラストは以下のような用途でご利用いただけます：
- プレゼンテーション
- ドキュメント
- ウェブサイト
- イベント資料
- チャットやメッセージでのリアクション画像
- ブログ記事の挿絵
- プルリクエストやコードレビューでのLGTMスタンプ
